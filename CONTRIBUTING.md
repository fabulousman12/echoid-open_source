# Contributing

Thanks for taking the time to contribute.

## Getting Started

1. Install dependencies with `npm install`.
2. Copy `.env.example` to `.env` and fill in your local values.
3. Copy `src/data.example.ts` to `src/data.ts` and adjust the local config if needed.
4. Run the app with `npm run dev`.

## Development Notes

- Keep changes focused and easy to review.
- Avoid committing secrets, platform service files, or generated artifacts.
- Prefer updating existing tests when changing behavior.

## Pull Requests

1. Create a branch for your change.
2. Explain the problem and the approach in the PR description.
3. Run the relevant tests before opening the PR.

## Security

If you find a security issue, please avoid posting exploit details in a public issue.
